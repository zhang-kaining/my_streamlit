# AI 对话助手 - Streamlit 应用

一个简洁、规范的 Streamlit 对话界面，用于与后端模型服务交互。

## 项目结构

```
.
├── app.py              # 主应用文件
├── config.py           # 配置文件
├── requirements.txt    # 依赖管理
├── .gitignore         # Git 忽略文件
└── README.md          # 项目文档
```

## 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 运行应用

```bash
streamlit run app.py
```

应用会自动在浏览器打开：http://localhost:8501
